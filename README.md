
## Usage
```bash
# install resource manager(Option)
$ npm i -g nrm
$ nrm use taobao

# web install and run
$ npm i
$ npm run serve
$ npm run build

# electron install and run
$ npm i
$ npm run electron-dev electron开发
$ npm run electron-package 打包成exe
```
